Forensics report:

For this report. The group gatherd and divided the tasks among us. I was tasked with researching the artifacts of the affected PC. The artifacts contained of: Disk image, Memory dump, pcap file. Here i swiftly found inside the pcap file suspicious traffic.

This image does not provide all information but you can see the protocol http being used no a non private ipadress. This ringed bells and i discussed it. After looking further into the IP adress there has been multiple connections made from that source IP adress. Most importantly, a download had been made from the same IP. the conclusion in here was that a succesful phising attack was conducted, wich convinced the user to download a executable named "Update tool". This is most likely a mix of Trojan and infostealer. since the actual malware was not trying to hide its intentions
