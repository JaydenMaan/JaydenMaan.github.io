Forensics report:

For this report. The group gatherd and divided the tasks among us. I was tasked with researching the artifacts of the affected PC. The artifacts contained of: Disk image, Memory dump, pcap file. Here i swiftly found inside the pcap file suspicious traffic.

This image does not provide all information but you can see the protocol http being used no a non private ipadress. This ringed bells and i discussed it. After looking further into the IP adress there has been multiple connections made from that source IP adress. Most importantly, a download had been made from the same IP. the conclusion in here was that a succesful phising attack was conducted, wich convinced the user to download a executable named "Update tool". This is most likely a mix of Trojan and infostealer. since the actual malware was not trying to hide its intentions but had a userfriendly name that would fool most not technical users.

autopsy sample of infected disk:

The malware was found on the device. Here we extracted it using Autopsy. !!!!!!!!!!!!autopsy!!!!!!!!!!! We searched for the name of the suspicious download and found the file. We anaysed the file using Ghidra and IDA Free 9.0. After some time we found out that it was an info stealer. It searched common filestructures(browser files/local directoy's) and extracted private RSA key's. !!!!!!!!!!!!!!!malware sample!!!!!!!!!!!!!!!!` It first encrypted all the information and then send it To a C2 server.
